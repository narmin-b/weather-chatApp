# Weather & Chat App

A SwiftUI-based iOS application that combines interactive weather mapping with real-time user chat.

## Key Features

1. **Interactive Weather Map**
   - Renders a global map using MapKit.
   - Zoom in/out controls.
   - Tap anywhere on the map to pin a location and view its real-time weather details (temperature, description, location name) in a custom presentation sheet.

2. **Authentication**
   - Firebase-backed user registration and secure login.
   - Persistent user sessions on device (UserDefaults storage for fast logins).

3. **Real-time Peer-to-Peer Chat**
   - Browse other registered users in the app.
   - Send and receive real-time messages via Google Firestore.
   - Live message streams automatically updating chat sessions.
